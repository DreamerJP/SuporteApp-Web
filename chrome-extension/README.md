# SuporteApp — Extensão Chrome

Versão extensão do [SuporteApp Web](https://dreamerjp.github.io/SuporteApp-Web/), com popup compacto e configurações em aba separada.

## Instalação (modo desenvolvedor)

1. Abra `chrome://extensions` no Chrome ou Edge.
2. Ative o **Modo do desenvolvedor** (canto superior direito).
3. Clique em **Carregar sem compactação** e selecione esta pasta (`chrome-extension`).
4. Fixe o ícone do SuporteApp na barra de ferramentas.

## Uso

- **Popup (clique no ícone):** botões compactos que copiam o texto com a otimização OPA. Campo de IP no topo — cole o IP (ou clique no botão de colar, que lê direto da área de transferência) e as abas HTTP/HTTPS + portas configuradas abrem automaticamente.
- **Configurações (engrenagem ou clique direito no ícone → Opções):** abre em uma aba com gerenciamento de textos (criar/editar/excluir), fonte de textos por URL, portas IP, otimização OPA, aparência e backup.

## Migração do app web

O backup é intercambiável: exporte o **backup completo** no app web (Config → Backup & Dados → Exportar) e importe nas configurações da extensão. O caminho inverso também funciona.
